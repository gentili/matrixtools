class NetworkConnection
	- Generates NetworkActivityInfo	objects to be delivered to

**** Data Classes

class NetworkInfo
	- network config data

class NetworkLoadInfo
	- some network stats that would be helpful in 
	guessing how heavy the network activity is on 
	this machine

class NetworkConnectionID
	- Unique identifier for a network connection

class NetworkConnectionInfo
	- Info for a single connection

class NetworkActivityInfo
	- an event class which encapsulates the amount
	and type of activity occuring on a NetworkConnection
